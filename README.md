## 1. 목적
이 _Github_ 의 목적은 아래와 같습니다.

1. R 데이터 분석의 개인적인 연습
2. 나처럼 시작이 막막한 사람들에 대한 가이드
3. 정보 공유를 통한 지식의 확장

## 2. 대상 독자
- 데이터 분석을 시작하고 싶으나 나처럼 시작이 막막한 사람

- 데이터 전처리, 시각화, 모델 프로세싱까지 Step by Step으로 따라해보고 싶은 사람

- 그냥 내가 좋은 사람 :)

## 3. 다루는 내용
1. 캐글을 처음  접근하는 방법
2. 타이타닉으로 천천히 데이터분석 시작하는 방법
    - 데이터 전처리 방법
    - 데이터 시각화
    - 모델링
    - 평가 방법
3. 사소한 꿀팁들
